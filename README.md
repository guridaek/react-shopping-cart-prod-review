# react-payments

우아한테크코스 레벨2 장바구니 미션

## [배포 페이지](https://react-shopping-cart-prod-git-guridaek-guridaek.vercel.app/)

## 패키지 설치

```
npm i
```

## 로컬 실행

```
npm start
```

## 스토리북 실행

```
npm run storybook
```

## MSW

- 장바구니와 주문 목록은 로컬 스토리지에 저장됩니다.
- 로컬 스토리지를 비워서 초기화할 수 있습니다.
- 쿠폰 목록은 4종류로 고정되어 있습니다.
